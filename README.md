# Sky High: Neon Sphere Challenge

우주 공간 속에서 펼쳐지는 스릴 넘치는 3D 플랫포머 게임입니다! 네온 빛으로 가득 찬 세상에서 정상까지 올라가 보세요.

## 🚀 게임 소개
당신은 네온 스피어(공)가 되어 끝없이 펼쳐진 공중 발판을 뛰어넘어 가장 높은 곳(정상800M)에 도달해야 합니다. 
도중에 만나는 이동하는 발판, 부서지는 발판 등 다양한 장애물을 극복하고, 체크포인트를 활용하여 정상까지 도전하세요!

## 🎮 조작법
- **이동:** `W`, `A`, `S`, `D` 또는 `방향키`
- **점프:** `Space`
- **시점 변경:** `마우스`
- **일시정지:** `ESC` (음악 제어 가능)
- **리스폰/재시작:** `R`

## 💡 게임 팁
- **체크포인트:** `깃발`이 꽂혀 있는 황금색 발판은 체크포인트입니다. 닿으면 진행 상황이 저장되어, 실패하더라도 저장된 지점부터 다시 시작할 수 있습니다.(깃발이 빨간색 -> 초록색으로 바뀌면 체크포인트 저장완료!)
- **Expert Zone:** 600m 이상 올라가면 발판의 크기가 작아집니다. 집중력을 발휘하세요!
- **부서지는 발판:** 발판 색상이 어둡게 변하면 곧 사라지니 주의하세요.
- **움직이는 발판:** 주황색으로 빛나는 발판은 좌우로 움직입니다. 타이밍을 잘 맞춰서 뛰어오르세요!
